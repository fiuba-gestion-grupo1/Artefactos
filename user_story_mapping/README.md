# User Story Mapping

## Descripción del artefacto

El User Story Mapping es un artefacto visual que organiza las historias de usuario en un mapa, mostrando de manera ordenada las funcionalidades y el flujo de uso del sistema, con el propósito de comprender mejor la experiencia del usuario, priorizar las tareas y planificar entregas de valor de forma clara y progresiva.

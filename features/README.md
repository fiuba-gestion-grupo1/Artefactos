# Puntaje de los Features

## Descripción del artefacto

El artefacto Features es una lista de funcionalidades propuestas para el sistema, en la que cada una se evalúa y puntúa con estrellas del 1 al 5 según su relevancia o valor, con el objetivo de priorizar de manera visual y sencilla qué características son más importantes para el producto y deben ser atendidas primero en el desarrollo.

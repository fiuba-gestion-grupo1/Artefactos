# Estructura de desglose del trabajo (WBS)

## Descripción del artefacto

La Estructura de Desglose del Trabajo (WBS) es una representación jerárquica que divide el proyecto en partes más pequeñas y manejables, organizando todo el trabajo en entregables o tareas específicas, con el fin de facilitar la planificación, el control y el seguimiento del proyecto de manera clara y ordenada.

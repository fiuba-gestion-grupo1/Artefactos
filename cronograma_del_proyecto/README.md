# Cronograma del proyecto

## Descripción del artefacto

El cronograma del proyecto es una planificación que organiza y representa en el tiempo todas las actividades, tareas y hitos de un proyecto, mostrando su secuencia, duración y dependencias, con el objetivo de dar claridad sobre qué se hará, cuándo se hará y quién lo hará, permitiendo controlar el avance y asegurar que el proyecto cumpla sus plazos.

# User Stories

## Descripción del artefacto

El artefacto **User Stories** forma parte de Lean Inception y se utiliza para **representar funcionalidades desde la perspectiva del usuario**.
Cada historia expresa una necesidad, acción y propósito concreto, permitiendo alinear el desarrollo técnico con el valor que espera recibir el usuario final.
Además, constituyen la base del Product Backlog, priorizando las entregas de forma iterativa e incremental.

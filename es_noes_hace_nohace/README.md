# Es - No Es - Hace - No Hace

## Descripción del artefacto

El artefacto **Es / No Es / Hace / No Hace** forma parte de Lean Inception y sirve para **delimitar el alcance del producto**.  
Permite definir claramente qué características forman parte de la identidad de la solución y cuáles no, ayudando a evitar confusiones con los stakeholders y alineando expectativas dentro del equipo.

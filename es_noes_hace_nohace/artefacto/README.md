# Es - No Es - Hace - No Hace

## Es

- Una plataforma digital de planificación y gestión de viajes.  
- Un espacio donde los viajeros pueden dividir y controlar gastos en grupo.  
- Una herramienta para realizar check-in online y ahorrar tiempo en hoteles.  
- Una comunidad donde los usuarios comparten experiencias y recomendaciones.  
- Un sistema que premia la participación con puntos y descuentos.  
- Una solución accesible y multiplataforma (web y móvil).  
- Una herramienta pensada para diferentes perfiles: estudiantes, familias, mochileros, ejecutivos.  
- Una plataforma que cubre el viaje completo: antes (planificación), durante (experiencia) y después (feedback).  

## No Es

- Una agencia de viajes que organiza paquetes turísticos.  
- Un servicio de transporte o aerolínea.  
- Una billetera digital o aplicación de pagos independiente.  
- Un CRM para hoteles o restaurantes.  
- Una simple copia de TripAdvisor centrada solo en reseñas y precios.  
- Un sistema exclusivo de reservas (no reemplaza a Booking ni Despegar).  
- Una red social genérica (el chat está enfocado solo en el viaje).  
- Un marketplace de turismo con paquetes cerrados.  

## Hace

- Permite planificar viajes de manera integral y colaborativa.  
- Divide gastos grupales de forma automática y clara.  
- Facilita el check-in online en hoteles para ahorrar tiempo.  
- Ofrece un sistema de recompensas por participación activa (reseñas, fotos, consejos).  
- Proporciona un chat para viajeros filtrado por destino, fomentando la conexión entre usuarios.  
- Presenta listas de actividades y lugares destacados en base a reseñas de la comunidad.  

## No Hace  

- No organiza paquetes turísticos cerrados.  
- No reemplaza los sistemas de gestión internos de hoteles o aerolíneas.  
- No funciona como billetera electrónica independiente para pagos.  
- No compite como red social generalista (su uso es exclusivo para el contexto de viaje).

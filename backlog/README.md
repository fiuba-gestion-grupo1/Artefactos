# Backlog

## Descripción del artefacto

El Backlog es una lista priorizada y en constante evolución que reúne todas las tareas, requisitos o mejoras pendientes en el desarrollo de un sistema, y su propósito es dar visibilidad y organización al trabajo, asegurando que el equipo siempre se enfoque en lo más importante y valioso para el producto o el negocio.

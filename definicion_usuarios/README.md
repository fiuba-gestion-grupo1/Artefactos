# Definicion de usuarios

## Descripción del artefacto

La definición de usuarios es el artefacto que describe de forma clara quiénes serán los usuarios o perfiles que interactuarán con el sistema, identificando sus características, necesidades y objetivos, con la intención de orientar el desarrollo hacia soluciones que realmente aporten valor y respondan a las expectativas de quienes usarán el producto.
